# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 프로젝트 개요

GitHub Pages를 이용한 개인 포트폴리오 사이트입니다. Jekyll 기반으로 자동 빌드되며, Markdown으로 콘텐츠를 작성합니다.

## 구조

- `index.md`: 실제 포트폴리오 콘텐츠 (이력서, 기술 스택, 프로젝트 경력)
- `index_b.md`: GitHub Pages 기본 템플릿 (참고용)
- `_config.yml`: Jekyll 설정 파일
- `data/`: 이미지 및 정적 리소스 (프로필 사진, 스크린샷 등)

## 콘텐츠 수정

포트폴리오 내용 수정 시 `index.md` 파일을 편집하면 됩니다. 커밋 후 GitHub Pages가 자동으로 사이트를 재빌드합니다.

## 이미지 참조

이미지는 `data/` 디렉토리에 저장되며, Markdown에서 `data/filename.png` 형태로 참조합니다.
